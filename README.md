# Mi App Docker 🐳

Aplicación web desplegada con Docker y Nginx.

## Requisitos
- Docker
- Docker Compose
- Git

## Pasos para desplegar

1. Clonar el repositorio
git clone https://github.com/ivanoviedo174-stack/mi-app-docker.git

2. Entrar a la carpeta
cd mi-app-docker

3. Ejecutar la aplicación
docker-compose up -d

4. Abrir en el navegador
http://IP_DEL_SERVIDOR

## Tecnologías
- Nginx (servidor web)
- Docker Compose
- HTML/CSS
